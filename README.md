# HarmonyOS-Watch-Network-Speed-Test-Demo



Installation requirements:
Install DevEco Studio • Set up the DevEco Studio development environment.The DevEco Studio development environment needs to depend on the network environment. It needs to be connected to the network to ensure the normal use of the tool.The development environment can be configured according to the following two situations 1.If you can directly access the Internet, just download the HarmonyOS SDK 2.If the network cannot access the Internet directly, it can be accessed through a proxy server • Generate secret key and apply for certificate
